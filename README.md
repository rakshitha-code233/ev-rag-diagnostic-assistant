# EV Diagnostic Assistant 🚗

This project is a Retrieval-Augmented Generation (RAG) based AI assistant for EV technicians.

## Features
- Uses EV manuals (PDF)
- Semantic search with vector database
- Local LLM using Ollama
- Streamlit web interface
- Provides answers with source references

## Tech Stack
- Python
- LangChain
- ChromaDB
- Ollama (Phi model)
- Streamlit

## How to Run
1. Install dependencies
2. Run `ollama run phi`
3. Run `streamlit run app.py`

## Example
Ask: "How to charge Tesla Model 3?"
